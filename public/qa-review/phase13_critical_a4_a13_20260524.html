A-4KB-A041 — PC-0335対象外品番リスト不完全(LE/DPF欠落)

カテゴリ: 基本/概要 / 基材活用 / ダイライト6FAL下地 | ペルソナ: 設計者
質問
3Mダイノックを大建工業の「ダイライト6FAL」下地に貼ると不燃材料になるのですか?施工方法は?
短答
ダイライト6FAL(大建工業の火山性ガラス質複合板・厚6mm)下地への3Mダイノック貼付は、AE/CA/CH/CN/FA/FE/FW/HG/LW/ME/MW/NU/OM/PA/PC/PG/PS/PT/RT/SI/ST/TE/VM/WG等の指定シリーズが大臣認定番号NM-5784(2)で不燃材料になります(2024年Ver.1基準)。重ね貼り不可、AR/DG/DR/EX/FLS/末尾TIL等は対象外です。
詳細本文
技術資料PC-0335-00(2024/11/1)に詳細記載。接着力データ(プライマー無し: 19.7N/25mm幅、DP-900N3塗布: 26.6N、WP-2000塗布: 39.6N等、目安10N以上で十分な接着性)。施工方法は2系統: 1)現場施工: 設置済みダイライト6FALに直接貼付。ジョイントは透かし目地(目地幅6mm程度)で、端部・小口・目地底にプライマー塗布、2)Dボード工法: 工場でダイライト6FALに四周巻き込みでフィルム貼付→パネル化→現場で両面テープ+接着剤で固定。裏面に粘着が付くなら3MプライマーDP-900N3塗布。失敗回避ポイント: AR(凹凸柄)、DG、DR、EX、PS-MTRC、FLS、末尾TIL、NEO-FLE、WH-111は対象外。重ね貼りすると不燃材料の対象外となる。
関連品番: 3MダイノックAE/CA/CH/FE/PA/PS/ST/VM等多数
対象メーカー: 3M / 対象ブランド: ダイノック
出典: 3M技術資料 PC-0335-00 2024/11/1「3Mダイノックフィルム:ダイライト6FAL下地への貼付について」
🚨 問題箇所
詳細本文の対象外リストに『LE-1551/1552』『DPF-100』が欠落(PC-0335 PDF公式リストには含まれる)。※Phase13検証W1 H-4の『DW-2474MT・DW-2481MT混入』指摘は本文に該当文字列なし=検証レポート側の誤指摘の可能性大
💡 主くろ修正案
対象外リストを PC-0335 PDF準拠で完全版に書き換え: LE-1551/1552 / PS-MTRC全品番 / AR / DG / DPF-100 / DR / EX / FLS / NEO-FLE / -TIL / WH-111(LE-1551/1552・DPF-100を追記)

A-5KB-B041 — Dilite=アルミ複合板 誤同定

カテゴリ: 下地別施工 / アルミ複合板 / Dilite/アルミ複合板への施工 | ペルソナ: 職人
質問
アルミ複合板(Dilite等)の下地に3Mダイノックを貼ることはできますか?サイン用途で検討しています
短答
アルミ複合板はサインの定番下地で3Mダイノック貼付可能ですが、屋外用途ならEX/EXRシリーズを選定します(屋内用一般シリーズは耐候性不足)。プライマーDP-900N3塗布でアルミ表面に十分な接着力を確保。ただし溶融亜鉛めっき鋼板やフッ素塗装は使用不可。EXRシリーズの場合、適用下地表(技術資料PC-0387)でアルミ=◎(垂直面10年耐候)です。
詳細本文
アルミ複合板(代表的にAlpolic/Dibond/Dilite=三菱ケミカルアルポリック等)はアルミ表板2枚で芯材をサンドした複合パネルで、サイン・看板・店舗外装で多用されます。3Mダイノック施工可否: 1)屋内用途: 一般シリーズ(AE/WG等)+DP-900N3プライマー、十分な接着力(アルミ下地接着力38.2N/25mm幅程度・PC-0385データ)、2)屋外用途: EX/EXRシリーズ必須(技術資料PC-0387適用下地表でアルミニウム=◎、垂直面10年耐候性)、焼付塗装鋼板/チタン/ステンレスも○、3)注意点: 塩ビ鋼板はMEシリーズ/PS-4007EXRで変色の可能性あり使用不可、溶融亜鉛めっき鋼板は湿度高環境で接着力低下で使用不可、無垢材/木材/モルタル/ガラスは×(端部から水分浸入で接着不良)、4)前処理: アルコール脱脂→#180サンディング→DP-900N3塗布(端部・小口は重ね塗り推奨)。失敗回避ポイント: アルミ複合板はエッジから水分が入りやすいため、屋外曝露面のエッジは必ず四周巻き込みまたはコーキング処理。Dilite等の表面塗装が劣化していると剥離するため、塗装状態確認必須。
関連品番: 3MダイノックEX/EXR/WDシリーズ(屋外)・一般シリーズ(屋内)
対象メーカー: 3M / 対象ブランド: ダイノック
出典: 3M技術資料 PC-0387「EXRシリーズ製品ブリーフ」適用下地表/PC-0385「WDシリーズ製品ブリーフ」接着力データ
🚨 問題箇所
短答・詳細で『アルミ複合板(Alpolic/Dibond/Dilite)』とDiliteを並列分類しているが、Dilite=ダイライト®6FAL=大建工業の火山性ガラス質複合板(不燃)で別カテゴリ
💡 主くろ修正案
Diliteをアルミ複合板列から除外。『ダイライト6FAL=大建工業の火山性ガラス質複合板(不燃NM-5784(2))・別カテゴリ』と注記分離

A-7KB-F005 — WB系=ホワイトボードシリーズ 品番創作

カテゴリ: 材料選び / ダイノック / WBシリーズ(ホワイトボード) | ペルソナ: 設計者・オフィス管理者
質問
ダイノックのWBホワイトボード機能シートで会議室の壁はどう変わりますか?
短答
既存壁を全面ホワイトボード化できる粘着剤付きシートで、ホワイトボードマーカーで書いて消せるようになります。会議室・教室・打合せスペースの壁を「書ける壁」に変えられ、専用ホワイトボード設置不要になります。
詳細本文
ダイノックのWBシリーズ(品番例:WB系の正式品番は要確認)はホワイトボードマーカーで筆記・消去可能な表面を持つ塩ビシートです。<br>(1)機能:ホワイトボード専用マーカー(油性ではなく水性マーカー)で文字や図を書け、専用イレーザーや乾いた布で消去できます。<br>(2)推奨用途:会議室壁面・スタートアップオフィスのアイデア出しスペース・学校の教室・塾の自習室・ブレインストーミングルーム等です。<br>(3)サイズ:壁一面(例:幅3m×高さ2.4m)を全面ホワイトボード化できるのが最大の強み。市販のホワイトボード(幅1.8m前後)では足りない大型空間に向きます。<br>(4)注意:油性マーカーで書くと消えなくなります。客先への引き渡し時に「水性ホワイトボードマーカーのみ可」と必ず伝達します。また、書いたまま長期放置するとゴーストイメージ(うっすら跡が残る)が出るため、使用後は早めに消すよう案内します。<<br>失敗回避ポイント:ホワイトボードシートは振動部・可動部(扉)NG・垂直壁面のみが推奨です(意匠系のWBシートは類似品多く、メーカー仕様要個別確認)。
関連品番: WB系全般(品番要確認)
対象メーカー: 3M / 対象ブランド: ダイノック
出典: 3Mダイノック施工マニュアル(2016改訂)12.シリーズ別注意・3Mダイノック製品ブリーフ_PB-DIN-001
🚨 問題箇所
『WB系/WBシリーズ』記述・3M公式に『WB』存在せず・品番インデックス0件=創作疑い。正規はWH(WH-111/WH-200)・PWF(PWF-500)・IWB(IWB-600)

A-8KB-F012 — ダブルカット重ね幅 5〜10mm誤

カテゴリ: 施工技術 / ダイノック / 継ぎ目処理 | ペルソナ: 中級職人/設計士
質問
ダイノックの継ぎ目は突き付け貼りできますか?重ね貼りの正しい手順は?
短答
ダイノックも他塩ビシートと同様、ジョイント突き付け貼りはNGです。必ず重ね貼り→カッターでまとめ切り(ダブルカット)→重ね部除去→密着の手順で処理します。屋外は端部シーリングも必須です。
詳細本文
塩ビシートのジョイント(継ぎ目)処理は施工品質を左右する最重要工程です。<br>(1)突き付け貼りNGの理由:塩ビシートは温度変化で微妙に伸縮するため、突き付け(隙間なく並べる)で施工しても数か月で目すき(隙間)が出ます。一度開くと水分・ほこりが入り、剥がれが進行します。<br>(2)正しい重ね貼り(ダブルカット工法):<br> 手順1:1枚目を貼った上に、2枚目を5〜10mm重ねて貼ります。<br> 手順2:重ね部の中央(両方のシートを貫通)にカッターで定規当てて一発切りします(これを「まとめ切り/ダブルカット」と呼びます)。<br> 手順3:不要な余白(下側1枚目の端材+上側2枚目の端材)を剥がします。<br> 手順4:継ぎ目部をスキージーとドライヤー加熱で完全密着させます。<br>(3)カッター刃の注意:切れの悪い刃で切ると下地まで傷つけたり切り口が雑になります。<br>失敗回避ポイント:大判の柄合せが必要な品番(SUPER REAL WOOD系・グラフィック系)は、ジョイント位置を意匠的に目立たない位置(柱の影・天井際)に設計段階から決めます。
関連品番: ダイノック全般
対象メーカー: 3M / 対象ブランド: ダイノック
出典: 3Mダイノック施工マニュアル / 共通/施工要領_共通フロー.md ⑤端部処理 / 既存KB-A013
🚨 問題箇所
『1枚目の上に2枚目を5〜10mm重ねて貼る』は3M公式マニュアル(2022/11改訂 P.13)『約20mm重ね・剥離紙30mm残し』と矛盾
💡 主くろ修正案
3M施工マニュアル2022/11改訂 P.13『約20mm重ね・剥離紙30mm残し』に修正

A-9KB-F024 — 出典PDF誤帰属(施工上の注意_R5 P.44)

カテゴリ: 施工技術 / ガラスフィルム / 水貼り工法 | ペルソナ: 中級職人/施工管理
質問
ガラスフィルムの「水貼り工法」とはどんな施工方法ですか?気泡やエッジスペースはどう処理しますか?
短答
水貼り工法はガラス面と粘着面の両方に中性洗剤を薄めた施工液を吹付けて貼る方法です。位置調整しながら貼れ、スキージーで水を中央から外側へ追い出すことで気泡を抜きます。フィルム端とガラス枠の間は2〜3mmのエッジスペースを必ず確保します(3M公式PDF明記値)。施工温度は12〜38℃が推奨です。
詳細本文
ガラスフィルム施工は塩ビシートと異なり「水貼り工法(ウェット工法)」が基本です。<br>(1)準備:中性洗剤(食器用洗剤等)を水で薄めた施工液をスプレーボトルに入れます。3M公式PDFには中性洗剤を水で薄めた施工液とだけ記載があり、具体的な配合比は明記されていません(<br>(2)施工温度:3M公式PDF「ガラスフィルム施工上の注意_R5」では推奨温度12〜38℃・直射日光があたらない時間帯での施工が指示されています。最低5℃以上が条件です。<br>(8)養生期間:総厚350μm超の防犯フィルムは1ヶ月程度の養生期間が必要(冬季はさらに延長)と3M公式PDF multimedia.pdf P.46に明記。通常フィルムの養生期間は3M公式PDFでは明文化されていません。<br>失敗回避ポイント:**エッジスペースは1〜2mmでなく必ず2〜3mm(3M公式PDF明記値)**。1〜2mmでは少なすぎて温度伸縮で端部が枠と干渉し剥離原因になります。施工温度12〜38℃を厳守し、酷暑・極寒・直射日光下での施工は避けること。
関連品番: 3Mファサラ・3Mウインドウフィルム全般
対象メーカー: 3M / 対象ブランド: 3Mファサラ/3Mウインドウフィルム
出典: 3M公式PDF「ガラスフィルム施工上の注意_R5」P.44 エッジカット項 /「ガラスフィルム_multimedia.pdf」P.46 養生期間
🚨 問題箇所
出典『ガラスフィルム施工上の注意_R5 P.44』だがR5はP.37-40のみ収録(P.44なし)・実際は製品ガイド_R4 P.44由来
💡 主くろ修正案
出典を『製品ガイド_R4 P.44(フィルムのエッジカット項)』に訂正(内容自体は正)

A-10KB-F021 — 出典PDF誤帰属(使用上の注意4_R2 JIS A 5759)

カテゴリ: 材料選び / ガラスフィルム / 飛散防止 | ペルソナ: 住宅オーナー・店舗オーナー
質問
3Mのガラスフィルム飛散防止品はどんな効果がありますか?JIS認定はありますか?
短答

# B26: スマホビュー 納品書トグル文字被り修正

- 完了日: 2026-04-14
- 担当: くろさん
- commit: 3dbd648

## 問題
スマホ幅(375px)で現場売上入力モーダルの「納品書」トグルをON(右)にすると、
トグルのサムが左隣の「納品書」ラベル文字に重なって読めなくなる。

## 原因
`label`内が `[納品書] [==○] [要/不要]` の3要素横並び。
ONにするとサムが右へ移動し、トグル幅(36px)が左の「納品書」テキストに被る。

## 修正内容
`src/components/SalesModal.tsx` を2箇所修正:

1. **入力エリア(524行目周辺)**:
   - 左の単独 `<span>納品書</span>` を削除
   - 右テキストを「要/不要」→「納品書要/納品書不要」に統合
   - gap を `gap-2` → `gap-1.5` に調整
   - button に `shrink-0` 追加

2. **既存エントリ(691行目周辺)**:
   - span に `whitespace-nowrap` 追加
   - button に `shrink-0` 追加

## 確認
- スマホ(375x812): ON/OFF両状態でテキスト被りなし
- PC(desktop 1280x800): レイアウト崩れなし
- フィールド構造・入力欄は変更なし（健太郎指示遵守）

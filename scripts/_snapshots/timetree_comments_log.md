## Supabaseへの登録方法

コメントを登録するには `scripts/patch_comments.mjs` を作成して実行:
```js
// .env.local の SUPABASE_URL / SUPABASE_SERVICE_ROLE_KEY を使う
// events テーブルで event_id をキーに comments フィールドをPATCH
```

詳細は `_memo_for_claude4.md` 参照。
